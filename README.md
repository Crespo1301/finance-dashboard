## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Crespo1301/finance-dashboard.git

# Navigate to project directory
cd finance-dashboard

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5173` in your browser.

## ğŸ’» Usage

1. **Select Transaction Type**: Choose between Income or Expense
2. **Enter Details**: Add description, amount, category, and date
3. **Add Transaction**: Click "Add Transaction" button
4. **View Analytics**: Check your summary cards, pie charts, and line graphs
5. **Set Budgets**: Create monthly spending limits for each category
6. **Search & Filter**: Use the search bar and filters to find specific transactions
7. **Edit Transactions**: Click the edit icon to modify existing entries
8. **Export Data**: Download your transactions as a CSV file
9. **Change Currency**: Select your preferred currency from the dropdown

Your data is automatically saved to localStorage and persists across sessions.
